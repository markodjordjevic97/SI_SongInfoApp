# SI_SongInfoApp
Ideja projekta:

Desktop aplikacija za pregled informacija o dostupnim pesmama.

 Administrator:
1. Logovanje
2. Unos pesme(naziv, žanr, izvođač..)
3. Izmene
 Korisniku će biti omogućeno:
1. Uvid u informacije o pesmama
2. Meni sa žanrovima
3. Sortiranje
4. Search (pretraživanje pesama)

 Napomena:
Aplikacija omogućuje korisniku uvid u sve informacije vezane za pesmu.
Baza podataka sa pesmama će biti na serveru (cloud) tako da će omogućiti brz odgovor na korisničke zahteve.
